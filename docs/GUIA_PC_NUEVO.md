# New PC guide

1. Download the latest `FutonHUB-Launcher.exe` from GitHub Releases.
2. Run it.
3. Enter a fine-grained GitHub token limited to `FutonEspaiHUB` with
   `Contents: Read-only`.
4. Select the authorized `.env` file when requested.
5. Wait while the launcher creates folders, installs the managed Python runtime
   and dependencies, downloads the ERP commit and performs the health check.
6. FutonHUB opens automatically.

Future ERP and launcher updates are checked automatically.
